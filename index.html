Ericsson Projects

Ericsson Emaps

Ericsson Emaps is a google maps API powered tool used by NBN and Ericsson to map satellite and fixed wireless installation appointments and their job progress. The project also calculates and shows tropospheric ducting on the australian continent for Ericsson networks.

Installation verification models

The processing of P12 and P11 business process artifacts was automated by developing a ML model. Formerly manually done by the BPT, an OCR and Image recognition model was developed to automate the auditing and verification of installations

EPS

The Ericsson Provisioning Suite project is made up of many microservices some of which I developed. Interfacing with specific customer DBs and our data lake.

Remote developer environment set-up

The investigation of remote developer requirements and the set up of remote environment images for Azure platform. Firewall networking and drive mounting.

LIFT

Automated dynamic ONVIF physical IP-based security camera device discovery and set up. Explicit NDA

Postgres DB to GCP migration

Converted all SQL scripts to BigQuery. Updated DB relationships.
